# Task 1: Rule-Based Chatbot

## Description
This project is a simple rule-based chatbot developed using Java.
The chatbot responds to user inputs based on predefined rules using
conditional statements.

## Features
- Responds to greetings
- Answers basic questions
- Handles unknown inputs
- Exit command to end conversation

## Technologies Used
- Java
- Conditional Statements
- String Matching

## How to Run
1. Compile the program:
- javac RuleBasedChatBot.java


2. Run the program:
- java RuleBasedChatBot



## Output
The chatbot interacts with the user through the console and provides
responses based on predefined rules.
