# Movie Recommendation System

## Description
This project implements a **Movie Recommendation System** where users
receive movie suggestions based on their preferred genre.
It is a simple **rule-based recommendation system** developed in Java
as part of the **CODSOFT Artificial Intelligence Internship**.

## Features
- Genre-based movie recommendations
- User-friendly console interaction
- Predefined movie database
- Case-insensitive genre matching

## Technology Used
- Java
- Object-Oriented Programming (OOP)
- Collections Framework

## How to Run
1. Compile the program:
   - javac MovieRecommender.java

2. Run the program:
   - java MovieRecommender
